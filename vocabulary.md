# Словарь терминов

- **ИИ (AI)** - искусственный интеллект (Artificial Intelligence)
- **LLM (Large Language Model, большая языковая модель)** - реализация искусственной нейронной сети, специально обученная понимать и генерировать текст. Пример: ChatGPT, Gemini, Llama
- **Artificial Neural Networks (ANN)** - математическая модель, предназначенная для обработки информации и решения задач машинного обучения
- **Generative AI (Генеративный ИИ)** - способен создавать текст, изображения, видео на основе обученных моделей
- **Hallucinations (Галлюцинации)** - ошибочные или вымышленные результаты (не соответствующие действительности), сгенерированные ИИ
- **Code Generation** - процесс создания кода с помощью ИИ
- **Synthetic Data** - искусственно сгенерированные данные для обучения моделей
- **Chatbot** - программа на основе AI для ведения диалога с пользователями
- **AI Assistants / Code Assistants** - программы на основе AI, которые помогают пользователю в выполнении задач
